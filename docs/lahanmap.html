Catatan kejujuran data. Batas kabupaten & angka per-wilayah di sini masih ilustratif untuk prototipe — menunggu polygon LSD resmi ATR/BPN (2021) untuk presisi. 3 titik petani adalah data nyata hasil wawancara lapangan di Banten; 156 titik jaringan memakai koordinat nyata namun beratribut sintetis, ditandai jelas.
